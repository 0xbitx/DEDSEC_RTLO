<h1 align="center">DEDSEC_RTLO</h1>

### DESCRIPTION
DEDSEC_RTLO is a simple Python tool that renames files using the Unicode RTLO (Right-To-Left Override) character to spoof file extensions. Itâ€™s commonly used to make an `.exe` file appear as a `.pdf`, `.jpg`, or any other extension by reversing how the filename is displayed in file managers.
